

## 
### 获取用户列表
**URL:** http://localhost:8080/user/list

**Type:** POST


**Content-Type:** application/json; charset=utf-8

**Description:** 获取用户列表

**Request-parameters:**

Parameter | Type|Description|Required|Since
---|---|---|---|---
name|string|姓名|true|-
subName|string|别名|true|-
id|string|编号|true|-
age|int32|年龄|false|-
ageI|int32|年龄|false|-
count|double|统计|false|-

**Request-example:**
```
curl -X POST -H 'Content-Type: application/json; charset=utf-8' -i http://localhost:8080/user/list --data '{
	"name":"昊强.方",
	"subName":"昊强.方",
	"id":"121",
	"age":18,
	"ageI":175,
	"count":23.92
}'
```
**Response-fields:**

Field | Type|Description|Since
---|---|---|---
name|string|姓名|-
subName|string|别名|-
id|string|编号|-
age|int32|年龄|-
ageI|int32|年龄|-
count|double|统计|-

**Response-example:**
```
[
	{
		"name":"昊强.方",
		"subName":"昊强.方",
		"id":"121",
		"age":18,
		"ageI":639,
		"count":94.17
	}
]
```
